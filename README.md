# 📚 BookSphere - Smart Library Management System

BookSphere is a full-stack Smart Library Management System designed to simplify library operations through an intuitive web interface. It enables librarians to efficiently manage books, members, and circulation while maintaining accurate inventory records.

🔗 https://booksphere-library-management-system.onrender.com

## ✨ Features

- 📖 Book Management (Add, Update, Delete, Search)
- 👨‍🎓 Member Management
- 📚 Book Issue & Return Management
- 📦 Automatic Inventory Updates
- ⏰ Due Date & Overdue Tracking
- 🔍 Search Functionality
- 📊 Dashboard with Library Statistics
- 📱 Responsive User Interface
- ☁️ Cloud Database Integration
- 🐳 Dockerized Deployment

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Ayyappan1204/BookSphere-Library-Management-System.git
```

### Navigate to Project

```bash
cd BookSphere-Library-Management-System
```

### Configure Database

Update `application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/library_db
spring.datasource.username=root
spring.datasource.password=your_password
```

### Run the Application

```bash
./mvnw spring-boot:run
```

Open:

```
http://localhost:9999
```
